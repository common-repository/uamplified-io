Connect your website with your uamplified.io account.

== Description ==

This plugin allows you to connect your uamplified.io products with your WordPress powered website.


== Installation ==

The plugin is installed just like any other WordPress plugin.
Once the plugin is activated, go to your Settings > uamplified.io page in the wp-admin area and enter your uamplified hosted websites domain.
Once you have saved your domain, you can start connecting your uamplified products by entering in each products API key.
After you have added your products, visit the Appearance > Widgets page to access your custom uamplified product widgets.


== Frequently Asked Questions ==

= Do I need a uamplified.io account to use this plugin? =
Yes, it is a minimum requirement.

= Can I use this plugin if my website does not use SSL? =
No. uamplified.io will not acknowledge any API calls that are not made over https!

= Where can I find my product API key? =
You can find it in your uamplified.io account on the Options > Integration tab for your product.
